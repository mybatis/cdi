简介

什么是 MyBatis-CDI？

MyBatis-CDI 扩展负责 MyBatis 映射器和 SqlSession 的生命周期。 MyBatis 组件直接注入 CDI 组件即可使用，无需创建或销毁它们。 它还提供本地和 JTA 事务支持，基于 @Transactional 注解。

环境要求

在开始使用 MyBatis 与 CDI 的集成框架之前，必须熟悉 MyBatis 和 CDI 的术语。本文档不提供 MyBatis 或 CDI 的背景信息或基本设置和配置教程。

MyBatis - CDI 需要 Java EE 6 或更高版本以及任意的 MyBatis 版本。
